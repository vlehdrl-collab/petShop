<h3>프로젝트를 하면서 느낀점</h3>

첫번째 팀프로젝트의 경우 기존에 있던 샘플을 가지고 레이아웃을 복사해서 팀원들끼리 새로운 홈페이지를 만드는 과정이었습니다
단순히 복사붙여넣기만하고 내용을 조금씩 수정하기만 하면되는 과정이었는데, 이런식으로 공부를 하면 머리속에 남는것도 없고
배끼기 밖에는 안될 것 같아서 제가 맡은 파트를 다한이후에, 저혼자 따로 책을보면서 전체적인 레이아웃을 다시 공부하였습니다.
당연히 시간이 좀 걸릴 수 밖에는 없었고, 또 제가 추가적으로 하기로 했던 부분을 다른사람이 해버리는 바람에 제가 할수있는게 별로 없었습니다.
이미 다른사람들이 다해버려서 구글폰트를 따와서 전체 폰트를 바꾸고 게시판의 굴꼴이나 모양을 만드는 정도밖에는 하지못했던것 같습니다.
다들 의욕만 너무 앞서있어서 욕심많은 사람이 더 많은 분량을 담당하면서 더 큰 목소리를 내고 조장이라는 사람이 제게 책을 보지 말고
이미 완성된 홈페이지를 수정하라고 해서 스트레스를 많이 받았던 것 같습니다.
궁극적으로 저는 저혼자 홈페이지를 만들수잇는 실력을 쌓는게 목적이었기 때문에 화가났지만 참으면서 이미 완성된 프로젝트에서 내가 할수있는 부분을 찾아서 햇던것 같습니다.
